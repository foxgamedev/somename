Сюди клади папку system/ (і будь-які інші змінені теки клієнта),
зберігаючи структуру клієнта. Приклад:

    client/system/skillname-e.dat
    client/system/sysstring-e.dat
    client/system/Interface.u

Після того, як поклав файли:
    python3 ../make_manifest.py . ../manifest.json   (або з кореня репо: python3 make_manifest.py client manifest.json)
    git add -A && git commit -m "system" && git push

Цей файл-нагадування можна видалити, коли покладеш system/.
